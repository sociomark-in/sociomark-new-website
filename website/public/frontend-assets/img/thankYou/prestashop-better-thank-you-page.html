This Prestashop Better Thank you page module by Knowband converts you order success page into a powerful marketing tool where you can show related products, automatically generated discount coupons, banners, and HTML content easily.

The Prestashop module replaces your old and static order success page with a new, dynamic and fully customizable thank you page. It will show all the information shown on the order success page, plus provide some more other interactions that will promote a repeat purchase from the same customer.

Benefits for Merchants:

  • With the help of this Prestashop custom thank you page, the admin can show reference number, related products on success page.
  • Admin can show banners on the success page with the help of this better thank you page module by Knowband.
  • The better thank you Prestashop addon allows the admin to display automatically generated coupon codes.
  • With the help of this Prestashop addon, the admin can display HTML content.
  • The Prestashop Better Thank you page addon offers the admin to manage sort order easily.

  • The Prestashop Better Thank you page module has the option to show/hide discount coupon on the success page.
  • The respective Prestashop plugin offers the option to show any type of HTML content.
